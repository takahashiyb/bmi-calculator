## Overview

### The challenge

Your challenge is to build out this body mass index calculator page and get it looking as close to the design as possible.

You can use any tools you like to help you complete the challenge. So if you've got something you'd like to practice, feel free to give it a go.

Your users should be able to:

- Select whether they want to use metric or imperial units
- Enter their height and weight
- See their BMI result, with their weight classification and healthy weight range
- View the optimal layout for the interface depending on their device's screen size
- See hover and focus states for all interactive elements on the page

Want some support on the challenge? [Join our community](https://www.frontendmentor.io/community) and ask questions in the **#help** channel.

### What I learned

I learned that it is better sometimes to work on smaller sections of the webpage first, then lay it out after as a whole. Also tried to change a lot of element's attributes using JS to properly set up for accessibility. Although the accessibility part still needs more improvement, I am glad that I got to work on my JS skills.

### Continued development

Learning about how to set up the accessibility is a work in progress, although, I got the gist of the topic, in testing, there will always be mistakes. I also need to learn to plan out the tasks as well as the design. Although I did it for this task, there were a lot of parts that I have overlooked which lead to more time wasted on reframing things that I have already done.
